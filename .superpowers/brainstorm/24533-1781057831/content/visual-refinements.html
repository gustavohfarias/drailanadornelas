✅ O que já está correto

🎯 Priorização dos Refinamentos

Revisão Completa (recomendado)

Inclui: Logos SVG + tipografia + espaçamentos + elementos decorativos + ajustes de cor

  • Alinhamento 100% com manual
  • Qualidade visual premium
  • Tempo: ~3-4h de implementação
B

Essencial — Ganhos Rápidos

Inclui: Logos SVG + hierarquia tipográfica + fixes críticos

  • Resolve principais problemas
  • 80% do impacto visual
  • Tempo: ~1-2h de implementação
C

Mínimo — Só Logos

Inclui: Substituição PNG → SVG no nav e footer

  • Resolve problema de performance
  • Ganho rápido (30min)
  • Deixa outros refinamentos para depois

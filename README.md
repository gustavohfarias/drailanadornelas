# Dra. Ilana Dornelas — Landing Page

Site institucional da clínica dermatológica Dra. Ilana Dornelas, Itaim Bibi, São Paulo.

## Estrutura

```
drailanadornelas/
├── index.html
├── assets/
│   ├── css/style.css
│   ├── js/main.js
│   └── images/
│       └── dra-ilana.jpg   ← adicionar foto aqui
└── README.md
```

## Foto

Adicionar a foto da Dra. Ilana em `assets/images/dra-ilana.jpg`.
Resolução recomendada: 900x1100px, formato JPG.

## WhatsApp

Número configurado em `assets/js/main.js` → variável `WA_NUMBER`.
Atual: 5511925243636

## Deploy

Hospedado via Cloudflare Pages conectado a este repositório.
